# Factors Affecting Students' Academic Performance

## Project Overview

This project analyzes factors affecting students' academic performance using R Programming.

The analysis includes:

- Data Import
- Descriptive Statistics
- Data Visualization
- Correlation Analysis
- Simple Linear Regression
- Multiple Linear Regression

## Dataset Variables

- Study Hours
- Attendance
- Sleep Hours
- Internet Hours
- Previous GPA
- Final Score

## Tools Used

- R Programming
- R Markdown
- dplyr
- corrplot

## Key Findings

- Study hours positively affect final scores.
- Attendance contributes to better academic performance.
- Previous GPA is a strong predictor of final scores.
- Regression models explain most variation in student performance.
